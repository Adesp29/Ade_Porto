About Me

Hai saya Ade Surya Pramudya,saya lahir di Kota Tegal pada tanggal 29 Juli tahun 2001,sekarang ini saya sedang mengenyam pendidikan di Sekolah Vokasi Universitas Gadjah Mada mengambil prodi D4 Teknologi Rekayasa Mesin.

Sekarang ini saya sedang tertarik dalam dunia design modelling, saya menekuni beberapa software design untuk mengasah kemampuan saya. Tidak hanya itu saya juga tertarik dalam dunia otomotif oleh karena itu sekarang saya bergabung dengan tim Yacaranda UGM untuk membantu pembuatan bodi mobil dan frame mobil tim Yacaranda.

Dalam bersosialisasi saya memilih untuk mengikuti beberapa event yang diadakan kampus, mengambil pengalaman dari event tersebut serta menambah teman dari berbagai fakultas. Semua kegiatan itu saya atur sedemikian rupa agar saya mendapatkan pengalaman serta sebuah batu loncatan untuk berbicara didepan umum.

My Project

Gambar pertama ini adalah salah satu desain rumah yang saya buat dengan software SketchUp

Desain helikopter ini adalah karya desain realistik yang dilakukan teman-teman satu kelas saya termasuk saya dengan barang asli berupa rc helikopter yang dibuat menggunakan software Solidworks.

Gambar diatas merupakan desain engine v6 yang saya buat menggunakan Solidworks dengan ukuran real untuk melatih skill saya.

Sebelum merambah ke dalam software solidworks,autocad, serta inventor saya terlebih dahulu menggunakan SketchUp untuk mendesain baik itu rumah ataupun benda lainnya seperti yang terlihat pada gambar diatas saya menggambar mobil tersebut untuk persiapan seleksi tim Yacaranda.

Sebelum mendesain helikopter dan mesin mobil saya terlebih dahulu mencoba menggambar drone untuk mencoba fitur-fitur yang ada pada software Solidworks.

Setelah mencoba beberapa software desain saya mulai terpikat dengan project analisis benda menggunakan ansys yang kebetuluan juga disediakan matkul wajib pada prodi saya, kita bisa mengetahui kelebihan dan kelemahan dari desain yang telah dibuat.
